# 🚀 DevConnect -- Developer Social Media Platform

> 🌐 A full-stack social networking platform built for developers to
> connect, share, and collaborate in real-time.

------------------------------------------------------------------------

## 🧠 Overview

DevConnect is a modern full-stack application that enables developers to
build their network, share content, and communicate seamlessly. This
project demonstrates real-world architecture including authentication,
REST APIs, real-time features, and scalable backend design.

------------------------------------------------------------------------

## ✨ Features

-   🔐 Secure Authentication (Clerk)
-   📝 Create, edit & delete posts
-   🖼️ Image uploads via ImageKit
-   📰 Dynamic feed system
-   💬 Real-time messaging
-   🔔 Live notifications
-   🤝 Connection / follow system
-   🔍 Discover users
-   👤 Profile & edit profile
-   📱 Fully responsive UI

------------------------------------------------------------------------

## 🛠️ Tech Stack

**Frontend** - React.js - Tailwind CSS - Axios - Clerk Auth

**Backend** - Node.js - Express.js - MongoDB (Mongoose)

**Integrations** - ImageKit (media storage) - Inngest (background jobs)

------------------------------------------------------------------------

## ⚙️ Installation

### 1. Clone Repo

    git clone https://github.com/koushikbajpayee06/SocialMedia-app
    cd devconnect

### 2. Frontend Setup

    cd client
    npm install
    npm run dev

### 3. Backend Setup

    cd server
    npm install
    npm run server

------------------------------------------------------------------------

## 🔐 Environment Variables

### Backend (.env)

    MONGO_URI=
    JWT_SECRET=
    IMAGEKIT_PUBLIC_KEY=
    IMAGEKIT_PRIVATE_KEY=
    IMAGEKIT_URL_ENDPOINT=

### Frontend (.env)

    VITE_CLERK_PUBLISHABLE_KEY=
    VITE_API_URL=
